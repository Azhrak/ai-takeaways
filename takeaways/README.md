# Takeaways

Takeaways are topic-oriented in this repository.

## Workflow

1. Capture quickly in [inbox.md](inbox.md).
2. Periodically move entries into the most relevant topic file.
3. Create a new topic file when no existing topic fits.

## Topics

- [Context management](context-management.md)
- [Grilling workflow failure modes](grilling-workflow-failure-modes.md)
- [Model selection for planning vs execution](model-selection-planning-vs-execution.md)

## Add a new topic

- Create a file in this folder using a focused topic name.
- Add the new link under Topics in this file.
- Add the topic to [README.md](../README.md) if it should be visible from root.
