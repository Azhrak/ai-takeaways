# AI Takeaways

A simple static notepad for useful AI takeaways I discover during:
- work
- free-time web browsing

## Quick Capture Template

Copy this block for each new takeaway:

```md
### YYYY-MM-DD — <short takeaway title>
- Source: <link or reference>
- Context: <work | free-time browsing>
- Takeaway: <what was useful>
- How I can apply it: <practical next step>
```
